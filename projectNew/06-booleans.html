Writepad Editor

WritePad Editor is an editor applicaiton that allows you create Web pages easily and quickly.

Step to open WritePad application:

  1. Click Start
  2. Click Programs
  3. Click WritePad

Advantages of using a WritePad editor:

  • It allows faster creation of Web pages
  • It allows easier updation of text or images in WhitePad editor
  • It is platform independen

Disavantages of using a WritePad editor:

  • It consumes more meory space.
  • It requires to fill-in more details suc as meta daate for each page.
